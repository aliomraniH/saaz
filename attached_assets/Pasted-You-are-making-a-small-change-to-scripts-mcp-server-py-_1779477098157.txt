How to rename in FastMCP

Simply rename the Python function. FastMCP registers the tool under the
function's name:

BEFORE:
@mcp.tool()
def list_tables() -> list[dict]:
...

AFTER:
@mcp.tool()
def saaz_list_tables() -> list[dict]:
...

Do this for all 4 generic tools. Leave get_artist, list_artists, and
search_artists unchanged.

Why this matters

mneme (the middleware layer at https://github.com/aliomraniH/mneme) routes
every tool call to an audit namespace using keyword matching on the tool name.
Without the saaz_ prefix, calls to list_tables / query / stats get logged
under "default" instead of "saaz_demo", making the audit history harder to
query by database. The routing keywords for saaz_demo include "saaz_" which
will match all renamed tools.
